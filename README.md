# Student Portal System (C++)

## Description

This is a console-based Student Portal System developed in C++.
It simulates a basic login system where multiple students can register and access their profiles.

## Features

* Student Registration (5 users)
* Unique ID validation
* Secure Login System
* Menu-driven interface
* View Profile
* Update Profile
* Change Password
* View Holidays
* Check Absents

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Standard Input/Output

## Functional Modules

### Registration

* Stores student details
* Ensures unique ID for each user

### Login

* Verifies ID and password

###  Menu Options

1. List of Holidays
2. Total Absents
3. Show Profile
4. Update Profile
5. Change Password
6. Logout

## How to Run

### Compile:

g++ main.cpp -o portal

### Run:

./portal
